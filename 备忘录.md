## 首次在本机建目录并拉代码

### 重要：在 GitHub 网页里点开 `.bat` 只会打开记事本，不会执行！

请用下面任一方式。

### 方式 A：下载单个脚本再运行（最简单）

1. 用浏览器打开并**下载**（不要只在网页里看）：
   https://github.com/fbi02042026/maoshugame/raw/main/本地初始化.bat
2. 在「下载」文件夹找到 `本地初始化.bat`
3. **双击**运行；若仍打开记事本 → 见下方「双击变成记事本」
4. 黑窗口跑完后，去 `E:\NewProject\maoshugame`，双击 **`index.html`** 试玩

### 方式 B：命令行（最稳）

1. 按 `Win + R`，输入 `cmd`，回车
2. 复制粘贴下面三行，每行回车一次：

```bat
mkdir E:\NewProject\maoshugame 2>nul
cd /d E:\NewProject\maoshugame
git clone https://github.com/fbi02042026/maoshugame.git .
```

3. 完成后打开文件夹 `E:\NewProject\maoshugame`，双击 **`index.html`**

### 方式 C：已有仓库时更新

在项目目录里双击 `tools\setup_local_project.bat`，或在 cmd 里：

```bat
cd /d E:\NewProject\maoshugame
git pull origin main
```

### 双击变成记事本怎么办？

1. `Win + R` → 输入 `cmd` → 回车  
2. 若下载了脚本，执行（路径按你实际下载位置改）：
   ```bat
   cd /d %USERPROFILE%\Downloads
   本地初始化.bat
   ```
3. 或不用脚本，直接用上面的「方式 B」三行命令。

### 没有 Git？

安装：https://git-scm.com/download/win  
安装时勾选 **Add Git to PATH**，装完重启 cmd 再运行。

## 日常：你怎么放新资源

1. 把压缩好的 PNG **按 `tools\asset_manifest.json` 里的文件名** 放进：
   ```
   E:\NewProject\maoshugame\assets\incoming\
   ```
2. 文件名对不上的，也可以先整包丢进 `assets\incoming\`，让 AI 帮你对照 manifest 重命名。
3. **不用自己改代码**，整理、缩放、覆盖 `assets\` 由 AI 在对话里完成。

## 提交规则（重要）

| 规则 | 说明 |
|------|------|
| **默认不提交** | 你往目录里放了新图，AI 只整理本地/云端工作区，**不自动提交** |
| **你说「提交」再提交** | 例如：「整理好了，提交一下」 |
| **提交 = 提交 + 推送** | 一说提交，就 `git commit` 并 **`git push` 到 GitHub**，两步一起做，不会只提交不推送 |
| **提交前 AI 应做** | 跑 `import` / `setup_assets` → `repack_assets_zip` → `git add` → `git commit` → `git push` |

一句话：**资源你往 `E:\NewProject\maoshugame` 丢；我整理；只有你让我「提交」时，才 commit 并 push 到 GitHub。**

## AI 整理素材时的标准流程

在仓库根目录 `E:\NewProject\maoshugame` 执行：

```bat
:: 方式 1：incoming 里按 manifest 名放好图后
python tools\setup_assets.py

:: 方式 2：你有一整文件夹压缩图（会覆盖 assets）
python tools\import_assets_folder.py --wipe "路径\到\你的png文件夹"

:: 方式 3：zip 包
python tools\import_assets_zip.py --wipe assets\incoming\compressed.zip

:: 打包 zip + 可选压缩大图
python tools\repack_assets_zip.py
python tools\optimize_pngs.py
```

整理完成后：**等你口令再** `git commit` + `git push`（一起执行）。
