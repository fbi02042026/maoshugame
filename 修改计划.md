# 肥猫莫追我 - 修改计划（v2 更新）

## 一、核心玩法流程（最新版）

```
启动 → 漫画(首次) → 【老鼠洞主界面】
                      │
                      ├─ 点击"出洞探险" → 标题"飞奔的奶酪"淡入淡出 → 进入第一个房间
                      │
                      │  【新手引导前两关】
                      │  引导1：房间1，收集食物→鼠洞直接结算（教基本操作）
                      │  引导2：房间1→房间2→鼠洞结算（教连续过关）
                      │
                      │  【正式跑酷 - 引导完成后】
                      │  进入房间 → 收集所有食物 → 回鼠洞
                      │       ↓
                      │  第一次回鼠洞时：画面闪烁 → 传送到一个新房间（不是回家！）
                      │       ↓
                      │  仙人出现："这可能是因为时光回溯出现的波动，导致老鼠洞和另一个房间
                      │           链接在了一起，赶快寻找出路吧，为师先走了"
                      │       ↓
                      │  继续在新房间收集食物 → 回鼠洞 → 又传送到下一个房间（随机）
                      │       ↓
                      │  累计传送5个房间后 → 鼠洞发金光 → 进入后真正回家结算
                      │
                      │  中途被猫抓住3次 → 失败，带着一半食物回家
                      │
                      ├─ 金币显示、商店入口、皮肤等
                      └─ 新手引导：仙人在各节点弹出对话指引
```

**关键变化：**
- **取消"门"的设计**，房间之间全部通过老鼠洞传送（鼠洞始终可见，不用等到收集完食物才出现）
- 引导完成后鼠洞变成"虫洞"，收集完食物进洞会传送到随机新房间，而不是直接结算
- 连续穿越5个房间后，鼠洞变金色才是真正出口
- 天赋精简为3种
- 加入趣味互动玩法（见第三节）

## 二、鼠洞传送机制详解

### 传送逻辑
- 每个房间鼠洞始终存在（位置按房间配置）
- **未收集够食物**进鼠洞：提示"还没拿到足够食物呢~"，不传送
- **收集够食物**进鼠洞：
  - 新手引导关：按引导流程走（引导1关直接结算，引导2关进下一房间）
  - 正式跑酷：
    - 第1~4次进洞：黑屏1秒 → 传送到随机新房间（不重复刚离开的房间），累计食物
    - 第5次进洞（累计穿越5间后）：鼠洞变金色发光，进洞 → 结算回主界面
    - 仙人台词在第一次传送后弹出
- 传送后状态保留：剩余生命、已选天赋、累计食物数
- 每个新房间猫重新从睡眠开始

### 视觉效果
- 传送时：屏幕闪白0.3秒 + 波纹效果（用Graphics简单实现）
- 第5个房间的鼠洞：金色呼吸光圈，提示"发光的鼠洞！这一定是回家的路！"
- 非出口鼠洞：正常深色洞

## 三、趣味玩法方案（多选，请勾选你想加的）

以下玩法都是**基于现有架构容易实现**、不破坏现有代码的：

### 方案A：推倒家具挡猫 ⭐推荐
- 某些家具（如架子、花瓶堆、堆高的箱子）旁边会出现"推"的提示
- 玩家靠近并按住偷取键1秒可以推倒家具
- 家具倒下后变成障碍物，猫被挡住需要绕路2-3秒
- 如果猫正好在旁边被砸中，眩晕2秒
- 每局随机2-3个可推倒家具
- **实现难度**：低，复用现有interactive家具标记和偷取进度条机制

### 方案B：滑板车道具 ⭐推荐（之前讨论过）
- 房间里随机刷新一个滑板车道具（红色圆点）
- 吃了之后移动速度+30%，碰撞不被抓但滑板车消失
- 碰到家具也会让滑板车消失
- 被猫追时的翻盘神器
- **实现难度**：低，现有PowerupItem已有toycar类型，只要改效果

### 方案C：金色食物（高风险高回报）
- 每个房间有1个金色食物，价值=3个普通食物
- 但拿到金色食物时会发出"叮"的一声，猫立刻被惊醒并向声音方向看
- 金色食物闪闪发光容易辨认
- 玩家选择：偷金食物快速凑数但容易被抓，还是稳扎稳打偷普通的
- **实现难度**：低，只是在FoodItem加个isGold字段

### 方案D：猫粮碗（调虎离山）
- 房间里固定有一个猫粮碗位置
- 猫巡逻时如果离碗很近，有概率走过去吃3-5秒（期间完全不追你）
- 玩家也可以主动撞一下碗发出声音，吸引猫过去
- 相当于玩家可以主动创造安全窗口
- **实现难度**：中，需要给CatAI加一个"eating"状态

### 方案E：毛线球（投掷诱饵）
- 拾取毛线球道具后，玩家可以朝摇杆方向"投掷"
- 毛线球落地后滚动，猫看到会追毛线球（像追激光笔一样）
- 持续4秒后猫才恢复正常
- 每局最多1个
- **实现难度**：中，需要实现投掷物和猫的新反应状态

### 方案F：关灯拉绳（视野博弈）
- 房间天花板有一根灯绳，玩家跳起来（或走过去）能拉到
- 灯灭后：猫的视野范围缩小60%，但玩家自己移动也稍慢（黑暗中摸索）
- 持续5秒后灯自动亮（或者猫会去重新打开）
- 适合高难度房间的策略玩法
- **实现难度**：中，需要全局暗化效果+猫视野参数调整

### 方案G：狗来了！（随机事件）
- 低概率触发：屋外传来狗叫，猫吓得钻到床底躲5秒
- 这5秒内完全安全，可以大胆偷食物
- 纯随机惊喜事件，增加变数
- **实现难度**：低，只是定时触发让猫进入隐藏状态

## 六、对话文案

### 鼠猫扯皮（进入房间时随机）
- 鼠："肥猫！今天你的食物我全包了！"
- 猫："ZZZ...嗯？有老鼠味..."
- 鼠："这猫睡得跟猪一样，嘿嘿嘿"
- 猫："又来？这次看你往哪跑！"
- 鼠："今天天气不错，适合偷奶酪~"
- 猫："我的食物...我的美梦..."

### 仙人引导
- 第一次进房间："小家伙，看到那些食物了吗？靠近并按住偷取！"
- 第一次偷到食物："很好！现在带着食物回鼠洞吧！"
- 第一次传送（关键）："这...这可能是因为时光回溯出现的波动，导致老鼠洞和另一个房间链接在了一起。赶快寻找出路吧，为师先走了！"
- 看到金色鼠洞："这鼠洞散发着金光！这一定是回家的路！"
- 被猫抓住："哎呀！小心那只肥猫！你还有命，快继续！"
- 生命只剩1条："只剩一条命了！小心行事！"
