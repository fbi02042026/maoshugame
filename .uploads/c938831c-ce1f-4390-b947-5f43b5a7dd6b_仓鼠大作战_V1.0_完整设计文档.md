## 1. 游戏概述

### 1.1 一句话玩法
玩家扮演小仓鼠🐹，趁猫咪🐱睡觉时放置陷阱引开它，偷取房间里的食物，带回鼠洞获得胜利。

### 1.2 核心定位
- 🎯 风格：休闲为主、思考为辅
- 📱 平台：手机竖版 9:16（微信/抖音小游戏 + TapTap + 原生 App）
- 🎮 视角：俯视 2D Q版软萌卡通
- ⏱️ 单局时长：60-90 秒

### 1.3 核心循环
```
开局长镜头（每关）→ 准备阶段 → 行动阶段（追逐）→ 回鼠洞 → 星级结算
     (3s)            (15s)         (60-75s)         (5s)
```

## 2. 核心循环详解

### 2.1 开局长镜头（每关都有）
| 时间 | 镜头 | 玩家 |
|---|---|---|
| 0-1.5s | 从鼠洞位置缩到全局地图 | 不可操作 |
| 1.5-2.5s | 平滑放大到最远目标物 | 不可操作 |
| 2.5-3.5s | 回到全局地图 | 不可操作 |
- 不可跳过

### 2.2 准备阶段
- 镜头在**全局地图**状态
- 玩家可操作：
  - 拖动地图查看各个角落
  - 点击地图放置陷阱（消耗能量）
  - 查看目标物位置（高亮/闪烁）
  - 查看鼠洞位置（发光）
  - 查看猫初始位置（😴 睡觉中）
- 屏幕底部中央：**[开始行动！]** 大按钮
- 老鼠在鼠洞里不可移动

### 2.3 行动阶段
- 点 [开始] → 镜头平滑回到老鼠身上（0.5s 过渡）
- **操作只有摇杆**（左下角）
- 镜头**始终跟随老鼠**
- HUD：食物数 / 血量 / 猫头指示器 / 警觉条 / 能量
- 镜头动态反馈：
  - 猫离老鼠远 → 镜头正常距离
  - 猫离老鼠近 → 镜头慢慢放大
  - 猫贴近 → 屏幕四周闪红边

### 2.4 回鼠洞
- 食物数 ≥ 目标数 → "回鼠洞"按钮绿色发光
- 玩家可选择：
  - **提前回洞**（带几个结算几个）→ 减少 1⭐
  - **全偷完再回**（带全部结算）→ 满分
- 鼠进洞瞬间播放"食物弹出堆洞口"炫耀动画

### 2.5 星级结算
- 0 命 = ❌ Game Over
- 1 命结束 + 全偷 = ⭐
- 2 命结束 + 全偷 = ⭐⭐
- 3 命结束 + 全偷 = ⭐⭐⭐
- 提前回洞 = 减少 1⭐

## 3. 视觉规则

### 3.1 家具/道具高亮
| 类别 | 类型 | 视觉 |
|---|---|---|
| 可利用家具 | 餐桌/沙发/茶几/箱子/餐车/餐椅 | **边缘金光** |
| 靠墙固定家具 | 冰箱/灶台/料理台/床/衣柜 | 普通外观 |
| 陷阱 | 香蕉皮/捕鼠夹/水桶/花盆 | 顶部面板高亮 |

- **对比清晰**：玩家一眼看出哪些能撞
- 撞过 3 次的家具**变灰**（V1 演示用，不做完整学习系统）

### 3.2 猫撞家具反馈
- 镜头**震动**（固定震感，不按家具大小变）
- 家具**抖动 0.3s**（不位移）
- 猫头顶**眩晕特效**（⭐ 星星 + 皇冠状特效，可换）
- 猫"喵~"叫声

### 3.3 镜头反馈
| 距离 | 反馈 |
|---|---|
| 猫远 | 镜头正常 |
| 猫近 | 镜头慢慢放大 |
| 猫贴近 | 屏幕四周闪红边 |

### 3.4 猫头像指示器
- 镜头里**没猫**时显示在屏幕**右侧边缘中央**
- 永远指向猫的真实位置
- 跟随猫的相对位置旋转/移动

## 4. 猫的机制

### 4.1 状态机
| 状态 | 行为 |
|---|---|
| 😴 睡觉 | 静止 10s → 慢慢醒来 |
| 🛋️ 慵懒 | 慢速挪动 8s → 进入巡逻 |
| 👀 半醒 | 直接进入正常巡逻 |
| 🚶 巡逻 | 慢速巡视房间各点 |
| ❓ 警觉 | 加速 0.75 倍 |
| 😾 追逐 | 锁定目标，1.35 倍速 |
| 🙀 被砸 | 撞到陷阱/家具，眩晕中 |
| 😿 愣神 | 丢失目标，原地发呆 2-4s |
| 🐈 溜达 | 慢悠悠走回自己的窝 |

### 4.2 警觉值系统
- 猫离得近 → 警觉值上涨
- 离开视线 → 警觉值下降
- 警觉值 > 30 进入警觉状态
- 警觉值 > 60 进入追逐状态
- 有**记忆效应**（被看到过会保持警惕）
- 愣神/溜达时警觉值下降加快（2.5 倍）

### 4.3 学习系统（统一机制，全局 3 次撞不晕）
| 撞过次数 | 眩晕时长 |
|---|---|
| 0 次 | 2.5s |
| 1 次 | 1.5s |
| 2 次 | 0.5s |
| **3+ 次** | **0s**（猫主动减速绕开） |

- **统一机制**：撞陷阱和撞家具用同一套机制
  - 视觉反馈统一（眩晕特效）
  - 学习系统统一（全局计数）
- **全局计数**：猫撞任何物体 3 次后，**所有物体都不晕**
- **关卡重置时清零**
- 撞过的家具**变灰** + 猫在该家具前**主动减速**

### 4.4 猫的初始状态（每局随机）
| 状态 | 概率 | 行为 |
|---|---|---|
| 😴 深度睡眠 | 40% | 静止 10s → 慢慢醒来 |
| 🛋️ 慵懒趴窝 | 40% | 慢速挪动 8s → 巡逻 |
| 👀 半醒巡逻 | 20% | 直接进入正常巡逻 |

### 4.5 猫的微反应
- 巡逻路径 ±15% 随机扰动
- 警觉值涨/降 ±10% 随机因子
- 玩家无法"完美背板"

## 5. 生命系统

### 5.1 基础规则
- 初始 **3 条命**
- 被猫撞 → -1 命
- 偷了东西被抓 → -1 命 + 掉 1 个食物
- 命数 = 0 → Game Over

### 5.2 食物规则
- 食物 = 经验/货币（结算时按数量给）
- 掉地上的食物**永远保留在地图上**
- 玩家可以回去捡
- **不会自动消失**（不做"3 秒不捡就没"的紧迫机制）
