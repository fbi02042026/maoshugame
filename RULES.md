# 仓鼠大作战 - 核心规则备忘录

> 以下规则为玩家反复强调的内容，每次修改代码前必须检查。

## 0. 铁律：每次改完代码必须测试
- **修改任何代码后，必须用浏览器打开测试，截图确认效果**
- 测试流程：打开页面 → 进入第1关 → 模拟猫追逐→撞家具→眩晕→醒来→再次追逐→第二次撞家具→不卡死→第三次撞家具→不卡死→抓到老鼠→猫困惑滑行→不卡死

## 1. 猫碰撞眩晕规则
- **只有全速追逐时（速度 >= 85% 最大速度）且距离较远（>80px）碰撞才会眩晕**
- 非全速/近距离碰撞：只推出，不眩晕
- 全速眩晕后减速到 50%，碰撞冷却 **0.05s**（短冷却让墙碰撞立即生效防止卡死）
- 非全速滑开：减速到 70%，碰撞冷却 0.1s

## 2. 猫不能卡死
- confused/stunned 状态时跳过 applyCatPhysics 的转向/避障，只沿当前方向滑行
- 家具碰撞 pushOut：眩晕时 r+20，非眩晕时 r+10
- pushOut 圆心在矩形内部时推向最近边
- 眩晕姿态：targetSpd = patrolSpeed * 0.4，targetVx=vx, targetVy=vy
- 眩晕结束后进入 patrol，alertValue 清零
- 困惑姿态：targetSpd = patrolSpeed * 0.3

## 3. 图片交替动画（自然不突兀）
- 停2~4秒 → 切另一张 → 停0.8~1.5秒 → 切回，随机间隔
- 猫暴怒：暴怒1/2 自然交替
- 猫警戒：警戒1/2 自然交替，停2.5~4.5秒
- 老鼠失败：失败1/2 自然交替

## 4. 猫图片素材
- 睡觉 → catSleep | 眩晕 → catStun | 吃惊 → catSurprise
- 警戒 → catAlert1/2自然交替 | 追逐 → catRage1/2自然交替
- 抓到老鼠/玩家失败 → catHappy
- 第一次撞到 → catAngry | 第二次撞到 → catRage1/2

## 5. 老鼠图片
- 正常 → mouse（鼠.png，保持宽高比）
- 失败 → mouseFail1/2 自然交替

## 6. 其他规则
- 老鼠速度 1.68，摇杆固定在屏幕底部，WASD+方向键
- 抓到老鼠：撞飞70px，无敌2.5秒，猫困惑1.5秒
- 第1关引导系统，猫被撞晕最多2次
