| `assets/scripts/cat/CatAI.ts` | 全面重写猫AI：新增行走/蓄能/弹射三态混合行为 |

### 猫AI行为规则
- **近距离（<150px）**：猫以90px/s速度行走追赶
- **远距离（>200px）或老鼠逃跑**：猫停止蓄能（40能量/秒，约2.5秒满）
- **弹射**：蓄满后直线发射，中途不可改变方向
- **碰撞规则**：
  - 碰墙反弹，速度降低30%
  - 碰家具眩晕1.5秒，速度降低80%
  - 累计3次眩晕触发愤怒模式（弹射距离+30%，碰家具不眩晕）
- **预判**：猫使用领先目标算法预测老鼠位置

### 登录动画序列
1. login_bg 直接显示
2. login_logo 果冻弹出（0→1.15→0.93→1.0，0.55秒，延迟0.3秒）
3. login_shu 从右侧飞入（0.68秒，quadOut，到位后±6px上下晃动）
4. login_mao 从右侧飞入（0.98秒，quadOut，到位后±5px上下晃动）
5. 按钮和底部文字淡入

### 底部协议区
- 勾选框（默认不勾选）+ "我已详细阅读并同意，用户协议 和 隐私保护指引"
- 用户协议和隐私保护指引为浅黄偏绿(#C8DC82)可点击链接
- 适龄提示12+图标
- 健康忠告文字
- 版本号 V1.0

### Toast提示系统
- 白字黑底，带黑色描边
- 停留2秒后上升淡出消失
- 用于未勾选协议时的提示

### 游戏流程更新
```
旧流程: Boot → (Comic) → Menu(登录+主界面) → Game
新流程: Boot → (Comic) → Login → Menu → Game
```

## 五、技术细节

### 坐标系统
- 所有UI元素基于Canvas(720,1280)，原点(0,0)=屏幕中心
- 节点位置使用设计分辨率相对坐标

### 安全规则
- 登录按钮添加防重复点击标志 `_enteringGame`
- 图片加载失败时跳过动画避免黑屏

### 按钮设计规范
- 主按钮：黑色背景+白色文字+圆角
- 交互反馈：SCALE缩放1.05~1.06
