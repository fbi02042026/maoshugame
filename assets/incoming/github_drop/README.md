# GitHub 网页上传 · 投放区

把新素材上传到 GitHub **对应子文件夹**即可，不用自己改名。  
上传完成后在 Cursor 里说：**「上传好了，帮我整理」**。

## 五个文件夹（和你电脑上一致）

| 文件夹 | 放什么 |
|--------|--------|
| [上面靠墙](./上面靠墙/) | 灶台、冰箱顶视图 → `top_*.png` |
| [靠墙](./靠墙/) | 侧墙家具 → `wall_*.png` |
| [随意](./随意/) | 沙发、桌子等中央大件 → `sofa_*`、`table_*` 等 |
| [装饰](./装饰/) | 花盆、木箱等小件 → `decor_*.png` |
| [食物](./食物/) | 坚果、草莓等 → `food_*.png` |

**皮肤**请继续上传到 `assets/skins/hamster/`（`hamster_skin_XX.png`）。

## 网页上传步骤

1. 打开 https://github.com/fbi02042026/maoshugame  
2. 点进上表里的文件夹（例如 `assets` → `incoming` → `github_drop` → `食物`）  
3. **Add file** → **Upload files**  
4. 把 PNG 拖进去 → **Commit changes**  
5. 回到 Cursor 跟我说一声

## AI 会做什么

拉取 → 改名 → 挪到 `assets/` → 打包 zip → 提交回 GitHub。
